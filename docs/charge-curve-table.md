# 蓄力曲线配表说明

策划只需使用 Excel、Numbers 或任意表格软件编辑 `assets/resources/config/charge_curve.csv`，保存为 CSV 后覆盖项目中的同名文件。不要删除对应的 `.meta` 文件。

## 表格格式

第一行固定为：

```csv
seconds,impulse
```

后续每行表示一个蓄力时间点和对应冲量。游戏会在相邻两行之间进行线性插值，因此玩家在任意时刻松手都能获得连续的力度变化。

## 校验规则

- 至少包含 2 行数值，最多 101 行。
- 第一行数值的 `seconds` 必须为 `0.0`。
- `seconds` 必须逐行增加 `0.1`，范围为 `0.0–10.0` 秒。
- `impulse` 范围为 `0.1–100`。
- 下一行 `impulse` 必须严格大于上一行。
- 空行和以 `#` 开头的注释行会被忽略。
- 表头、列数或任意数值不符合规则时，游戏会在控制台报告具体行号，并自动使用内置安全曲线。

要延长最大蓄力时间，继续以 `0.1` 秒间隔添加新行即可；要修改已有时间点，直接修改该行的 `impulse` 数值。
